# Sentiment-Monitoring-Tool-for-Palantir-Coverage
This is a tool that accepts web articles and scrapes their text to then analyze via a transformer model to determine overall sentiment and theme. The tool is specifically adjusted for articles related to Palantir.
The model used for classification is facebook/bart-large-mnli.
Code can be ran manually by downloading and running in powershell terminal through Streamlit.
Or, use this link to access the tool online via Streamlit: https://sentiment-monitoring-tool-for-palantir-coverage-3n8mserdgum4n3.streamlit.app/
